Week 17 (May 27th - May 31th)

At the beginning of this week, I started by testing the 2 new exercises uploaded to RADI 4.5.13, 3D Reconstruction and Localized Vacuum Cleaner. Both worked correctly, although the Localized Vacuum Cleaner exercise has code errors that prevent it from functioning as it should.

Additionally, I added a scroll to the tutor's window so that if the response was very long, it wouldn't take up too much screen space. I also styled the scroll to make it more cohesive with the website.


Throughout the week, I researched tools for measuring optimization such as RTF, htop, VGLRun, etc., to make progress in that area.

Once I understood how to use all these tools, I created a cost table where I measured the percentage of CPU, RTF, and FPS used by each exercise without graphics acceleration, on both Mac and Windows. Then, I transferred all this data to a table in Canva.

Finally, as every week, I updated the blog with another week's content.
